## Features

- Installs Desired SWAP With EZ
- Panel Migrations
- Fixed 0 CPU and 0 RAM Bug
- Beginner Friendly
- Create Databases For Pterodactyl
- Perform Wing Migrations
- Perform Daemon Backups
- Latest Build 
- Perform Wings and ServerFiles Backups

A Simple step To use it
